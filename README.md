# 🌐 YuanZhiYuMing's Personal Website

## ✨ 简介

这是我的个人网站，基于 [GitHub Pages](https://pages.github.com/) 托管。在这里你可以了解关于我的信息、查看我的项目以及获取联系方式。

## 🔗 访问地址

- **在线预览**: https://yuanzhiyuming.github.io

## 🛠️ 技术栈

- HTML5 / CSS3 / JavaScript
- GitHub Pages 静态托管

## 📁 项目结构

```
YuanZhiYuMing.github.io/
├── index.html    # 主页文件
└── README.md     # 项目说明文档
```

## 🚀 本地运行

1. 克隆仓库：
   ```bash
   git clone https://github.com/YuanZhiYuMing/YuanZhiYuMing.github.io.git
   ```

2. 进入项目目录：
   ```bash
   cd YuanZhiYuMing.github.io
   ```

3. 直接在浏览器中打开 `index.html` 文件即可预览。

## 📝 更新与部署

1. 对 `index.html` 或其他文件进行修改
2. 提交更改并推送到 `main` 分支：
   ```bash
   git add .
   git commit -m "更新内容"
   git push origin main
   ```
3. GitHub Pages 会自动构建和部署，几分钟后即可在线查看更新。
